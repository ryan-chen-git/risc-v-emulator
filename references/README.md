# Datapath reference diagrams

CS61C RISC-V datapath diagrams — the visual source of truth for the datapath
viewer (`web/datapath.html`, generated by `web/render.py`). Component box sizes,
shapes, names, wire routing, and label positions are matched against these.

Add the two images here (right-click the diagram in chat → "Save image as…"):

- `single-cycle-datapath.png` — the single-cycle layout the viewer reproduces
- `5-stage-datapath.png` — the 5-stage pipelined version
